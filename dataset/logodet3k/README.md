# LogoDet-3K

Logo detection dataset — 604 categories, 31,266 annotated images. Used as the primary training dataset for both models.

**Download:** [LogoDet-3K on Kaggle](https://www.kaggle.com/datasets/lyly99/logodet3k)

After downloading, extract here:

```
dataset/logodet3k/
├── images/
│   ├── train/
│   ├── val/
│   └── test/
└── labels/
    ├── train/
    ├── val/
    └── test/
```
