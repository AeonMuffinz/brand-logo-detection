# Weights

Trained model weights for YOLOv7 and RTMDet (pre-trained on LogoDet-3K + fine-tuned on the custom dataset).

**Download:** [Google Drive](https://drive.google.com/drive/folders/13lxC41ULiAe6bFJGp3Znt5EEgWU9T2B3)

Expected files after download:

```
weights/
├── yolov7_logodet3k.pt        ← YOLOv7 trained on LogoDet-3K
├── yolov7_finetuned.pt        ← YOLOv7 fine-tuned on custom dataset
├── rtmdet_logodet3k.pth       ← RTMDet trained on LogoDet-3K
└── rtmdet_finetuned.pth       ← RTMDet fine-tuned on custom dataset
```
